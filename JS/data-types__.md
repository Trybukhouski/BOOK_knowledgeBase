# DATA TYPES

# About
- Существует 8 типов данных.
- 7 типов данных - примитивы:
  - number
  - bigInt
  - string
  - symbol
  - boolean
  - null
  - undefined
- Непримитивный тип данных - object.
- Примитивные типы данных - иммутабельны.
