# OBJECT METHODS

### OBJ METHODS >> Rules >> Declaration
- Методы в объекте реализованы через функции, которые присвоены свойствам этого объекта.

  ```
  {
    hello: function() {
      console.log('hello')
    }
  }
  ```
  - Сокращенная форма записи:

    ```
    {
      hello() {
        console.log('hello')
      }
    }
    ```
